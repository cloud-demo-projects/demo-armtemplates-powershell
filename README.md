# Project Details
Sample project explaining process of deploying Azure resources using ARM templates, Powershell scripts and Azure DevOps CICD.

## Prerequisites
### Local Development
- VS Code
- Extensions
- Powershell Core 7

### CICD
- Azure DevOps Account
- Azure Subscription
- Service principle with appropriate rights
